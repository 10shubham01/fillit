About Fillit
//

Text you reuse, one // away.

Save snippets once. Type // in any text box on any site to drop them in. Everything lives on your device — no account, no sync.

  1. 1
    Create a snippet

    Give it a //shortcut and your text.

  2. 2
    Type // anywhere

    A picker appears as you type. ↑↓ to choose, Enter to insert.

  3. 3
    Use variables

    Wrap words in { }. {date}, {time}, {clipboard}, {cursor} fill in for you; anything else like {name} asks you on insert.
